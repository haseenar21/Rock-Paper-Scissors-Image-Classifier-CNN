# 🪨📄✂️ Rock Paper Scissors Image Classification using CNN

A Deep Learning project that classifies images of **Rock**, **Paper**, and **Scissors** using a Convolutional Neural Network (CNN) built with TensorFlow and Keras.

---

## 📌 Project Overview

This project trains a Convolutional Neural Network (CNN) on a Rock-Paper-Scissors image dataset. The model learns visual features from images and predicts the correct gesture.

---

## 🚀 Features

- Image classification using CNN
- Image preprocessing
- Data augmentation
- Model training and validation
- Prediction on unseen images

---

## 🛠️ Technologies Used

- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook
